- **書名**: 多元宇宙 (Plurality)

## 結構概覽

本書分為 7 大部分、30 餘節：

- **0 導讀**: 名家推薦、關於作者群、找到你的道路、致謝
- **1 序言**: [[數位觀照]]
- **2 資訊技術與民主**: 日益擴大的鴻溝 / [[玉山視野]] / 數位民主的日常
- **3 多元宇宙**: 何謂多元宇宙 / 活在⿻世界 / 相連的社會 / [[失落之道]]
- **4 自由**: 權利、作業系統與數位自由 / [[身分與人格權]] / 結社與多元公眾 / 商業與信任 / 財產與合約 / [[近用]]
- **5 民主**: [[協作技術與民主]] / 後符號交流 / 沉浸式共享實境 / 創意協作 / [[擴增審議]] / [[自適應行政]] / ⿻投票 ([[多元投票]]) / [[社會市場]]
- **6 影響**: 從⿻到現實社會 / 職場 / 健康 / 媒體 / 環境 / 學習
- **7 展望**: [[政策]] / 結語

## 中文版獨有的詞源優勢

本書序言 (1 數位觀照) 強調 — 在華語脈絡中，「[[數位]]」具有**雙重意涵**: digital + plural。「也有『plural』的意涵，由共融共創的傳統所滋養」。這是中文版獨有的詞源優勢，反映了文化特殊性與技術民主化的統一。

## 結語對台灣讀者的特別呼籲

> 「人們會把臺灣視為⿻的明燈和象徵，對於⿻的繁榮而言，臺灣就像以色列之於猶太人，或烏克蘭之於歐洲自由那樣重要」

## 名家推薦摘要

- **蔡英文總統**: 「過去民主是兩個價值觀的對決，臺灣民主已是不同價值觀的對話」
- **第十四世達賴喇嘛**: 「以積極的行動讓共善的種子破土而出」
