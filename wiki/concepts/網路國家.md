# 網路國家

## 定義

**網路國家** (Network State) ── Balaji Srinivasan 在 2022 年同名書中提出的概念:

> 「先在線形成一個高度凝聚力的數位社群, 透過加密貨幣建立經濟基礎, 最終購買物理土地獲得主權, 形成新的『網路國家』」

## 在 Plurality 框架中的位置

Plurality 將 Network State 列為「**[[自由意志主義]]**」(放任主義) 路線的代表性威脅:

| 路線 | 中心化方向 | 代表性威脅 |
|---|---|---|
| [[技術專家治理]] (專家統治) | 集中化 | [[監控資本主義]]、AI 寡頭、Big Tech |
| **[[自由意志主義]]** (放任主義) | 反社會 | 加密資產投機、**網路國家** |

## 批判要點

| 批判 | 詳細 |
|---|---|
| **逃離既有民主制度** | 主張「**退出**」(exit) 而非「**改革**」(voice) |
| **加密貨幣為基礎** | 認可加密技術的 [[自由意志主義]] 政治化 |
| **將領土主權商品化** | 「以資本購買主權」邏輯違反共同治理 |
| **同質化群體假設** | 主張結成「**高度凝聚力的數位社群**」 ── 忽視內部多元性 |

## 「網路國家」vs「[[數位民主]]」

| Network State | [[數位民主]] (Plurality) |
|---|---|
| 退出既有國家 | 改革既有民主 |
| 加密貨幣為主 | 多元機制 ([[多元投票]]、[[平方融資]] 等) |
| 私有產權絕對化 | 多元產權結構 |
| 同質凝聚 | 多元交織 |
| Balaji Srinivasan | [[唐鳳]] + [[衛谷倫]] |

## 「網路國家」與 [[去中心化自治組織|DAO]] 的關係

- DAO 是 Network State 的「**輕量版**」(無實體領土)
- 從區塊鏈技術發展出來
- Plurality 框架接受 DAO 作為**多元結社**的一種, 但拒絕 Network State 的「**主權商品化**」企圖

## 與「Crypto」相關的 Plurality 立場

Plurality 對 cryptocurrency / Web3 採**精細區分**:

- 接受: 區塊鏈作為**互通性技術** (interoperability)
- 接受: DLT 作為**特定協作場景**的工具
- **拒絕**: 加密貨幣**取代法幣**的願景
- **拒絕**: Network State 作為**取代民主國家**的願景

cf. 本書 4-3 章 [[商業與信任]] 對 Bitcoin 的批判。
