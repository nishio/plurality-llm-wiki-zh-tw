# 中央集權

## 定義

**中央集權** (Centralization) ── 將決策、權力與資源集中於少數中央機構的政治、經濟或組織體制。

對立概念: **去中心化** (Decentralization)、**分散** (Distribution)、**多中心** (Polycentric)。

## 在 Plurality 框架中的位置

中央集權是 [[一元原子論]] 在政治制度上的具體實現:

| 一元原子論 | → | 中央集權 |
|---|---|---|
| 個人 + 社會整體二極結構 | | 個人 + 國家二極結構 |
| 忽略多元交織群體 | | 削弱地方自治、社團、社群 |
| 簡化為普遍規律 | | 簡化為集中決策 |

## 與「**集中化威脅**」的關係

本書 2-0 章將「集中化」列為 [[技術專家治理]] (專家統治) 路線的核心特徵:

> 「大型 AI 模型的權力, 掌控在工程師小團體手中」 (2-0 章)

→ 「中央集權」傳統意義 (政治) + 「集中化」新意義 (技術) ── 二者在 Plurality 框架中相互強化。

## 對照 [[數位]] / [[多元]] 的反命題

Plurality 的核心反命題:
- 「[[多元]]」反「中央集權」── 多元交織取代二極結構
- 「[[數位]]」(數+位) 暗含「**多位**」── 反對單一中心

[[失落之道]]章列舉「**封包交換取代集中式交換機**」── 技術上的中央集權批判。

## 與 zh-tw / ja 字形差

字形差異: 「權↔権」── **意義完全相同**, 漢字 bridge 通過。

對照 [[國家]]、[[反社會]]、[[資本主義]]、[[超模性]] 也展現相同 zh-tw / ja 字面共通性。

## 具體歷史脈絡

中央集權在中文政治史中:
- 秦始皇統一六國 → 中央集權的最早系統化
- 唐宋至清 → 「**皇權集中**」、官僚體系
- 民國時期 → 五權憲法、地方自治嘗試
- 中華人民共和國 → 黨政高度中央化
- 中華民國 (台灣) → 從威權中央集權轉向**多元民主**

→ Plurality 概念在華語區具有「**對抗本土集中化傳統**」的文化意義。
