# g0v

## 概述 (出自 [[數位民主的日常章]])

- **成立**: 2012 年
- **發起人**: [[高嘉良]] (clkao) 等
- **精神**: 「**放碼過來**」
- **地位**: 「全球前 3 的知名公民黑客社群」

## 名稱由來

- 「g0v」 = 「**零時政府**」 (gov → g0v，把 o 換成 0)
- 暗示「政府版本 0.0」「從零打造的政府」
- 也暗示與美國 .gov 域名的對比

## 重要實績

### COVID 口罩地圖 (2020)
- 公民黑客**吳展瑋**等透過 g0v 平台號召眾包
- 「超商口罩地圖」→ 政府主動邀集合作 → 「藥局口罩地圖」
- 政府與公民協作的典範案例

### [[太陽花運動]] 支援 (2014)
- 直播、轉播、記錄
- 公民媒體角色

### vTaiwan
- Pol.is 平台的台灣應用
- 政策審議的協作工具

## 與政府的關係

g0v 維持「**不為政府工作，為公共利益工作**」的獨立性，但與政府協作頻繁。[[唐鳳]] 本身是 g0v 社群成員，後進入政府。

## 與其他國際公民科技社群

- en: Code for America
- ja: Code for Japan
- jp 「シビックテック」社群與 g0v 有交流
